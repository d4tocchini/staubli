03 - The Deliervable

app ▸ web ▸ print

A Desktop App that generates a web site designed to be printed

The desktop app alleviates any need for external cloud services, is more secure & eases the delivery of the design technology in an incapsulated whole. A web site is generated, as opposed to a pdf or likewise, as the eco-system of web-technology is of supreme utility in not reinventing the wheel.

04 - Input Mechanism

The product data & top-level copy - the meat of the rendered catalog - will be input in a semi-structured JSON/YAML-like data format

To minimize scope & time to delivery, the scope of this initial project will not include a human-friendly GUI for content management. Though, the data will be fully dynamic, and the data format fairly simple, updates will require some experience with JSON or similar markup syntax. In a follow up project, a graphical interface can be readily added atop the data format and enable anyone with MS Office exposure to do the dirty work.
